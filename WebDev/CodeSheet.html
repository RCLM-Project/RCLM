Coding Languages Cheat Sheet

One common problem in Digital Humanities projects is not that people with the correct skill sets don't exist, but rather that it is unclear what skills will be necessary for a project to succeed. For those without coding or web development experience, it can be extremely difficult to translate the plan of a project into functional skills needed to bring that project to completion. This coding cheat sheet is one way that the RCLM team hopes to help solve this problem.

This coding cheat sheet gives you a general idea of what each coding language is used for, which other languages are commonly used in unison with others, and the types of projects that each coding language would be used to complete. While this information is by no means comprehensive, we hope that it will be useful in defining the goals of your project and which skills will be needed to meet those goals.

HTML

For many digital humanists, HTML is the most recognizable coding language. HTML is not only powerful, it is also fairly intelligible for even new coders because of its recognizable tagging system.
Common Uses:
  • Creation of website
  • Minimal text formatting (like the text on this page)
  • Linking from one website or webpage to another
Commonly Used With:
  • CSS

CSS

While CSS is typically not required, it provides a lot of functionality during website creation. Many of the things that you would code with CSS can be done in HTML as well, but any highly specialized or intricate style details must be coded in CSS.
Common Uses:
  • Provides a style guide for HTML code on websites
  • Can be used to add all aesthetic features to a website
Commonly Used With:
  • HTML

Javascript

Javascript is commonly known in the digital humanities world for how counterintuitive it is. Unlike HTML and CSS, Javascript is not intelligible to new coders or those unfamiliar with the language because of its complex syntax and abbreviation system.
Common Uses:
  • Add function to websites
    • Create popup
    • Payment Information
    • Complex buttons
    • Accessibility features like being able to change the screen colors
    • Etc.
  • App development
  • Website development
Commonly Used With:
  • HTML and CSS for Website Development
  • Softwares for app development, etc.

Java

While many people use Javascript and Java interchangably, they are in fact different coding languages. You can think of Java as the more complicated (but more powerful) sibling of Javascript. Java can do many of the things that Javascript can as well (but not all).
Common Uses:
  • Complex Application Development
  • Android Applications
  • Scientific Computing
  • Advanced scripting
Commonly Used With:
  • In isolation

Python

Python has recently taked the coding world by storm and is now the worlds most used coding language because it can be used in so many ways. If you are looking for a utility code, Python is a great option not only because of its flexibility but also because it works well with many other languages.
Common Uses:
  • Website development (on most platforms)
  • Data science
  • Data visualization
  • Game design
  • Natural language processing (NLP) and artificial intelligence(AI)
Commonly Used With:
  • In isolation

R

Like python, R has become a recent phenom in the coding world. While often not considered by digital humanists because of its roots in data science, there are actually many functions of R that work well with digital humanities projects. R is also commonly discussed using the name R Studio because of the application in which many people run R called R Studio.
Common Uses:
  • Data Science
  • Data visualization
  • Working with text and text mining
  • Living documents and data sharing
Commonly Used With:
  • In isolation

Command Line Coding (Shell Coding)

Shell coding is one of the often forgotten skills that many projects will need at one time or another. The good news is that many of the coding languages described above require at least some knowledge of command line coding even just to install and set up applications used commonly with those languages. On Macs, command line coding is run out of the terminal and is a way of giving commands to your computers operating system. Unlike the other coding languages which most frequently function on websites, in games, or in applications, command line coding actually "codes" your operating system. There are many kinds of command line coding. In fact, command line coding is not really a type of code or language, just a common way to refer to a family of coding languages. Instead, the coding languages used on the command line are Shell Languages. The most commonly used shell language is bash (Bourne Again Shell), but other common shell languages include zsh and tcsh. You can also use many of the other coding languages discussed previously in the command line. In fact, the command line is a particularly common way to run Python scripts. All the information here is about bash, but much of it is applicable to all shellcode.
Common Uses:
  • Accessing OS operations
  • Accessing programs and downloads to use other coding languages
  • Setting up complex directories and repositories for projects
Commonly Used With:
  • Python
  • GUIs or applications for all other coding languages
Honorable Mention:

GitHub

While not required for projects, and while used by humanists more widely than any coding languages, many projects might require version control. For lots of teams, the web developer would also take a lead role in creating and manintaingng these version control systems. Though only one option, GitHub is easily the most common and familiar version control system used today. A nice feature of GitHub is that it is usable out of a website browser (though I don't recommend it), a desktop GUI, or the command line. For complex operations, you will need knowledge of using Git in the command line, so if your project will use GitHub this is something to ask potential team members and web developers about.
Common Uses:
  • Version Control
  • Sharing code, data, files, etc.
Commonly Used With:
  • All coding languages
  • Both code based and non-code based projects
